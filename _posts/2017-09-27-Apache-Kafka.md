# Apache Kafka
> A high-throughput distributed messaging system

* LinkedIn에서 개발된 분산 메시징 시스템 
* 확장성(Scale-out)과 고가용성(High availability)을 위하여 클러스터로 구성

## 기본 구성 요소와 동작 

* 발행/구독(PUblish/Subscribe) 모델을 기반으로 동작
* Producer, Consumer, Broker로 구성
